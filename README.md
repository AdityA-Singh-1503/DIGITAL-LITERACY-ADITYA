## Project Overview

This project is part of the Digital Literacy course and focuses on understanding how to use digital tools effectively, communicate professionally, and stay safe online.

The aim of this project is to build awareness about responsible digital usage and to develop basic skills required in today’s digital world.

---

## Task Summary

### Task 1 – Awareness Infographic

An infographic was created using a design tool to explain key topics like digital literacy, useful tools, and safe internet practices. This helped in presenting information in a simple and visual way.

---

### Task 2 – Digital Portfolio

Profiles were created or updated on platforms such as GitHub and LinkedIn. This task helped in understanding the importance of maintaining a professional online identity.

---

### Task 3 – Platforms Exploration

A beginner-level coding problem was solved on an online platform, and a Google Form quiz was created. This task introduced practical tools for learning and collaboration.

---

### Task 4 – Email Etiquette

Professional emails were written and a checklist for proper social media behavior was prepared. This improved understanding of formal communication.

---

### Task 5 – Cybercrime Awareness

A case study on cybercrime was explored along with a prevention checklist. This task helped in learning about online risks and safety measures.
